Full-Stack ML App
with Streamlit + Scikit-learn
An end-to-end machine learning web application enabling non-technical users to train, evaluate, and deploy classification models through a clean browser interface — no code required.
Why Built
To democratize ML experimentation — data teams needed a rapid prototyping tool without standing up a full MLflow/Airflow stack for every experiment.
Key Challenge
Balancing model retraining latency with Streamlit's session state model — stateless HTTP re-renders conflicted with long-running scikit-learn fit() calls.
Solution
Implemented @st.cache_resource for model persistence across sessions and joblib serialization for on-demand model persistence to disk.
Streamlit Scikit-learn Python pandas joblib
